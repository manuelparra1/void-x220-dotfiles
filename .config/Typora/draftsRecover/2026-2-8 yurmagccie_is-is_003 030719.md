**IS-IS. Part 3. Default and external prefixes**  
[IS-IS. Part 4. Wide-metrics](https://yurmagccie.wordpress.com/2018/10/23/is-is-part-4-wide-metrics/)

By default, the default route is injected into L2 LSDB. The default route can also be injected into L1 LSDB with additional configuration.

In order to inject default route into LSDB, the following configuration is used:

```
ip route 0.0.0.0 0.0.0.0 Null0
!
router isis 1
 default-information originate
```

Topology:

![IS-IS2.draw_io-IS-IS2.2.Areas](./images/is-is2-draw_io-is-is2-2-areas.jpg)

The default route is injected as internal prefix:

```
R5#sh isis database detail R5.00-00 | ex PID|Area|Add|IS

Tag 1:

R5.00-00            * 0x0000000C   0x05E7                 972/*         0/0/0
  Hostname: R5
  Metric: 10         IP 10.1.15.0 255.255.255.0
  Metric: 10         IP 10.1.25.0 255.255.255.0
  Metric: 10         IP 10.1.35.0 255.255.255.0
  Metric: 10         IP 10.1.45.0 255.255.255.0
  Metric: 10         IP 192.168.0.5 255.255.255.255
  Metric: 0          IP 0.0.0.0 0.0.0.0
```

In non-backbone area, the default route is injected with this configuration:

```
router isis 1
 default-information originate route-map rm-isis-default
!
ip route 0.0.0.0 0.0.0.0 Null0
!
route-map rm-isis-default permit 10
 set level level-1
```

Verification:

```
R6#sh isis database detail R6.00-00 | ex PID|Ar|Add|IS

Tag 1:

R6.00-00            * 0x0000000A   0x771B                1199/*         0/0/0
  Hostname: R6
  Metric: 10         IP 10.0.6.0 255.255.255.0
  Metric: 10         IP 10.1.26.0 255.255.255.0
  Metric: 10         IP 10.1.67.0 255.255.255.0
  Metric: 10         IP 192.168.0.6 255.255.255.255
  Metric: 0          IP 0.0.0.0 0.0.0.0
```

**External prefixes injection**  
This is straightforward, the only thing you need to specify is LSDB level if it is not backbone:

```
router isis 1
 redistribute eigrp 1 level-1
```

Verification:

```
R6#sh isis database detail R6.00-00 | ex PID|Area|Add|IS

Tag 1:

R6.00-00            * 0x0000000B   0xDC8C                1183/*         0/0/0
  Hostname: R6
  Metric: 10         IP 10.0.6.0 255.255.255.0
  Metric: 10         IP 10.1.26.0 255.255.255.0
  Metric: 10         IP 10.1.67.0 255.255.255.0
  Metric: 10         IP 192.168.0.6 255.255.255.255
  Metric: 0          IP 0.0.0.0 0.0.0.0
  Metric: 0          IP-External 6.6.6.6 255.255.255.255
```
